Operations about user

User

Get Access Token

POST /v1/GetAccessToken

A registered user can get access token. It requires client's IP address to be white listed. This access token is auto expiring, it expires after 1 hour (3600 seconds) from time of issue. (For reference: https://firebase.google.com/docs/auth/admin/create-custom-tokens#create_custom_tokens_using_the_firebase_admin_sdk)
